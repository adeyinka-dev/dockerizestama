

# STAMA - Student Accommodation Management Application

Welcome to **STAMA**, a revolutionary application designed to streamline the management of student accommodations, offering unparalleled ease-of-use and powerful functionality to both administrators and students.

## Overview 🌍

This application aims to simplify hostel and room management for administrators, while providing students with a seamless dashboard to view their accommodation details and raise maintenance requests. At the heart of STAMA lies a focus on user-friendliness and an intuitive design, ensuring everyone can make the most of its features without a steep learning curve.

## Features 🚀

- **Hostel and Room Management:** Enables administrators to oversee and manage various hostels and their respective rooms. 

- **Student Dashboard:** A dedicated space for students, offering them access to their personal details, room information, and a functionality to raise repair requests. 

- **Maintenance Requests:** Integrated system for students to report maintenance issues. Admins can view these issues, track their progress, and mark them as resolved.

- **Dynamic Status Updates:** Maintenance issues aren't just binary; track them as they move from 'Pending' to 'In Progress', through to 'Pending Inspection', and finally, 'Completed'.

- **Sleek UI:** A contemporary, intuitive user interface ensures that even non-tech-savvy users can navigate and use the application with ease.

## Setup  🛠

1. Clone this repository to your local machine.
2. Ensure you have Django and the necessary dependencies installed.
3. Navigate to the project directory and run:
   ```
   python manage.py runserver
   ```
4. Access the application through your browser, typically at `localhost:8000`.

## Usage 🖥

- **Administrators:**
  - Navigate to the admin dashboard.
  - Here, you can add, modify, or remove hostels and rooms.
  - The maintenance section provides a holistic view of all repair requests across hostels.

- **Students:**
  - Log into your dedicated dashboard using your credentials.
  - View your personal and room details.
  - Notice an issue? Raise a repair request directly from the dashboard.
